 - **SpringBoot**
 - **Ktor**

 - **RabbitMQ**
 - **Grafana**
 - **Prometheus**
 - **Loki**
 - **Promtail**
 - **CI/CD**
 - **Docker**

## Установка
1. Клонировать репозиторий
```bash
git clone https://github.com/Leturgone/MusicPlaylistsMicroservice.git
```
2. Cd к директории
```bash
cd MusicPlaylistsMicroservice
```
3. Запустить контейнеры
```bash
docker-compose up
```
