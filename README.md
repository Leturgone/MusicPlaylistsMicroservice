## Tech Stack

 - **Kotlin**
 - **SpringBoot**
 - **Ktor**
 - **PostgreSQL**
 - **RabbitMQ**
 - **Grafana**
 - **Prometheus**
 - **Loki**
 - **Promtail**
 - **CI/CD**
 - **Docker**

## Instalation 
1. Clone repository
```bash
git clone https://github.com/Leturgone/MusicPlaylistsMicroservice.git
```
2. Cd to directory
```bash
cd MusicPlaylistsMicroservice
```
3. Execute containers
```bash
docker-compose up
```
